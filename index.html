C'KADO

Plateforme de vente de chèques cadeaux en ligne pour le petit commerçant, vous pouvez trouver des commerçants par nom adresse ou ville et avec la géolocalisation. Génération de pdf et envoi de courrier automatiquement. Tableau de bord pour les commerçants de voir les ventes, les factures, les statistiques et de promouvoir leur module de vente sur les réseaux sociaux. Développé avec le modèle MVC: framework Slim 2.0 et Eloquent, méthode payement avec Braintree (Paypal compagny) et MangoPay. Actuellement c-kado.fr est en ligne.

Articom, 2016 HTML 5, CSS, jQuery, PHP, MySql

SportNet

SportNet est un projet scolaire développé par groupe de quatre, c'est une plate-forme pour gérer l'événement sportif, Il ya deux types d'utilisateurs: les participants et l'organisateur. Les participants peuvent s'inscrire seul ou avec un groupe pour des événements et voir leurs résultats. L'organisateur peut créer des événements et suivre les enregistrements, et bien d'autres fonctionnalités. Développé avec le modèle MVC: framework Slim 3.0, Eloquent et Twig.

LP CISIIE projet PHP, 2016 HTML 5, SCSS, jQuery, PHP, MySql

Une fois dans le channel, les utilisateurs peuvent envoyer des messages dans le chat. Une version de bureau a été généré grâce à electron
